## Installations Steps
 

## Backend
start installations packages node_modules
`yarn`

`yarn dev`

`yarn queue`

## Frontend
start installations packages node_modules
`yarn`

`yarn start`

## Mobile
start installations packages node_modules
`yarn`

start terminal Bundle
`yarn start`

start application
`yarn dev`
